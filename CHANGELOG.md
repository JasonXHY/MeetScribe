# 版本变更记录

> 格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)

## [0.9] - 2026-06-11

### 新增
- 音色库页面：独立管理页，支持查看、编辑、删除说话人，录音朗读添加声纹
- 声纹嵌入向量持久化：转写完成后保存到磁盘，重启后可恢复
- AI 设置多厂商支持：小米、百度、阿里等 11 家国内大模型
- API Key 明文/密文切换按钮
- 关于区域显示制作者信息

### 优化
- 文件列表列头对齐：表头与数据在同一 grid 中，列宽天然对齐
- 文件列表滚轮修复：递归绑定子控件，鼠标在任何位置都能滚动
- 录音按钮状态修复：停止后按钮文字正确恢复为"开始录音"
- 录音文件命名简化：`26061116会议.wav` 格式
- 列权重调整：文件名列缩窄，主题列加宽
- 主题显示：移除 20 字截断限制，显示完整内容
- 版本号统一：topbar、sidebar、关于区域使用同一常量

### 修复
- 修复文件列表高度不自适应窗口大小
- 修复声纹嵌入向量在程序重启后丢失
