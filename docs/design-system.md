# 侧耳倾听 v2.0 UI 设计规范

> 设计方向：Windows 11 Fluent Design 风格 — 干净、现代、克制
> 目标：专业的会议录音转写工具，舒适耐看，信息密度适中

## 1. 色彩系统

### 主色板

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg` | `#F8F9FA` | 页面背景 |
| `--surface` | `#FFFFFF` | 卡片/面板背景 |
| `--surface-hover` | `#F3F4F6` | 卡片 hover |
| `--border` | `#E5E7EB` | 边框/分隔线 |
| `--border-focus` | `#3B82F6` | 聚焦边框 |

### 文字色

| Token | 色值 | 用途 |
|-------|------|------|
| `--text-primary` | `#111827` | 标题/主要文字 |

| `--text-tertiary` | `#9CA3AF` | 占位符/禁用 |

### 功能色

| Token | 色值 | 用途 |
|-------|------|------|

## 5. 组件规范

### 5.1 按钮

**主按钮（Primary）**
- 背景：`--accent`，文字：白色
- 高度：32px，padding：0 16px
- 圆角：6px
- hover：`--accent-hover`
- 用于：主要操作（添加文件、保存设置）

**危险按钮（Danger）**
- 背景：`--error`，文字：白色
- 高度：32px，padding：0 16px
- 圆角：6px
- hover：`--error-hover`
- 用于：危险操作（开始录音、删除）

**次要按钮（Secondary）**
- 背景：透明，边框：1px solid `--border`
- 文字：`--text-secondary`
- 高度：32px，padding：0 16px
- 圆角：6px
- hover：背景 `--surface-hover`
- 用于：次要操作（合并转写、清空列表）

**图标按钮（Icon）**
- 背景：透明
- 尺寸：28x28px
- 圆角：4px
- 图标色：`--text-secondary`
- hover：背景 `--surface-hover`

**所有按钮规则**：
- 同级按钮统一 font-weight（都不加粗，或都加粗）
- 同级按钮统一高度
- 按钮组间距：8px

### 5.2 输入框（LineEdit）

- 高度：32px
- padding：0 12px
- 边框：1px solid `--border`
- 圆角：6px
- 背景：白色
- 聚焦：边框色变 `--border-focus`
- placeholder 色：`--text-tertiary`

### 5.3 下拉框（ComboBox）

- 高度：32px
- padding：0 12px，右侧留 28px 给箭头
- 箭头：**SVG data URI 小三角**（不使用 CSS border 三角）
- 背景：`#F9FAFB`（比白色略深，区分于输入框）
- hover：背景 `#F3F4F6`
- 下拉列表：白色背景，选中项背景 `--accent-light`
- 下拉项高度：32px，padding：0 12px

### 5.4 复选框（CheckBox）

- 尺寸：16x16px
- 未选中：边框 `#D1D5DB`，背景白色
- 选中：背景 `--accent`，内嵌白色对勾（QPainter 绘制）
- hover：边框色变 `--accent`

### 5.5 卡片（Card）

- 背景：白色
- 边框：1px solid `--border`
- 圆角：8px
- 内 padding：16px
- 内部行间距：8px

### 5.6 表单行（Form Row）

- 标签：固定宽度 80px，右对齐，颜色 `--text-secondary`，字号 13px
- 标签与控件间距：12px
- 行高：自动（由控件高度决定）
- 使用 `QFormLayout` 实现自动对齐

## 9. MiMo Code 实施要点

### 必须遵守的规则

1. **布局用 QFormLayout**（设置页表单）或 **QGridLayout**（文件列表），不要用 QHBoxLayout 手动拼接
2. **所有图标用 QIcon + QPainter**，不要用 emoji
3. **ComboBox 箭头用 SVG data URI**，不要用 CSS border 三角
4. **CheckBox 选中态用 QPainter** 画对勾，不要用 SVG data URI
5. **全局样式写在 styles.py** 的 `MAIN_STYLESHEET` 中，不要写 inline style
6. **同级按钮统一 font-weight 和高度**
7. **日志过滤**：`USER_FRIENDLY_KEYWORDS` 必须在 `_log()` 和 `GUILogHandler._poll_queue()` 中使用
