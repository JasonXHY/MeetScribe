## 2.2 AddVoiceDialog — 完整迁移旧版录音弹窗

**决策：完整实现旧版 AddVoiceDialog 的全部功能，不删减。**

7. **保存按钮**：提取成功后启用，调用 `library.add_speaker(name, embedding, "manual_recording", quality=0.90)`
8. **取消按钮**
9. **临时文件清理**：弹窗关闭时删除录音临时目录

**关键实现要点**：

```python
class AddVoiceDialog(QDialog):
    PRESET_TEXT = "你好，我是{姓名}，这是我的声纹样本。"

    def _start_recording(self):
        """开始录音"""
        import tempfile
        self._temp_dir = tempfile.mkdtemp()
        self._recorder = UnifiedRecorder(
            save_dir=self._temp_dir,
            use_vb_cable=False,
        )
        self._recorder.on_stop_complete = self._on_recording_complete
        self._recorder.start("mic")

    def _on_recording_complete(self, saved_files):
        """录音完成回调"""
        if saved_files:
            self._audio_path = saved_files[0]
            # 后台线程提取声纹
            thread = threading.Thread(target=self._process_audio, daemon=True)
            thread.start()

    def _process_audio(self):
        """提取声纹"""
        from voiceprint import VoiceprintLibrary
        library = VoiceprintLibrary()
        embeddings = library.extract_embedding_from_file(self._audio_path)
        if embeddings:
            self._embedding = list(embeddings.values())[0]
            # 启用保存按钮，更新状态

    def _save(self):
        """保存"""
        library.add_speaker(name, self._embedding, "manual_recording", quality=0.90)

    def closeEvent(self, event):
        """清理临时文件"""
        if hasattr(self, '_temp_dir') and os.path.exists(self._temp_dir):
            import shutil
            shutil.rmtree(self._temp_dir, ignore_errors=True)
        super().closeEvent(event)
```

### 2.3.2 音色库匹配建议

旧版在 `_add_match_suggestion` 方法中，对每个未命名的说话人做声纹匹配，如果匹配到已有音色，显示 "可能是 XXX (85%)" + "接受" 按钮。

```python
def _add_match_suggestion(self, row_layout, speaker, name_entry):
    """音色库匹配建议"""
    if speaker.get('name'):
        return  # 已有姓名，不显示建议

    spk_id = speaker['spk_id']
    embedding = self._speaker_embeddings.get(spk_id)
    if embedding is None:
        return

    try:
        from voiceprint import VoiceprintLibrary
        library = VoiceprintLibrary()
        matched_name, score = library.match(embedding)
    except Exception:
        return

    if not matched_name:
        return

    # 显示建议
    confidence_pct = int(score * 100)
    suggestion_lbl = QLabel(f"可能是 {matched_name} ({confidence_pct}%)")
    suggestion_lbl.setStyleSheet(f"color: {C_ACCENT}; font-size: 11px;")
    row_layout.addWidget(suggestion_lbl)

    accept_btn = QPushButton("接受")
    accept_btn.setFixedSize(40, 20)
    accept_btn.setStyleSheet(f"background-color: {C_ACCENT}; color: white; border: none; border-radius: 4px; font-size: 10px;")
    accept_btn.clicked.connect(lambda: name_entry.setText(matched_name))
    row_layout.addWidget(accept_btn)
```

参考旧版 `src/gui/dialogs.py` 第 523-576 行。

## 2.5 日志过滤 — 必须启用

**决策：启用 USER_FRIENDLY_KEYWORDS 日志过滤，与旧版行为一致。**

**核实结果**：旧版 `src/gui/home_page.py` 第 810-813 行确实使用了日志过滤：

```python
def _append_log_widget(self, message):

    if not any(keyword in message for keyword in USER_FRIENDLY_KEYWORDS):
        return  # 跳过不关心的日志

    ...
```

**新版修复方式**：在 `home_page.py` 的 `_log` 方法中添加过滤逻辑：

```python
def _log(self, msg):
    """添加日志（带过滤）"""
    # 过滤：只显示用户关心的日志
    if not any(keyword in msg for keyword in USER_FRIENDLY_KEYWORDS):
        return
    ts = datetime.now().strftime("%H:%M:%S")
    self._log_area.appendPlainText(f"[{ts}] {msg}")
```
