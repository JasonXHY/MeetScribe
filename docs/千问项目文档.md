## 一、项目概述

**侧耳倾听**（原名 MeetScribe）是一款 AI 会议录音转写工具。

核心功能：
- 录音（麦克风/系统音频/双轨）
- 转写（FunASR SenseVoice + CAM++ 说话人分离 + ct-punc 标点恢复）
- AI 摘要与纠错（MiMo 云端）
- 多格式输出（md/html/txt/srt/json）
- 音色库管理（声纹比对、录音朗读添加、持久化）

---

## 二、迁移背景

**原因**：
1. 原版使用 customtkinter（基于 Tcl/Tk），UI 流畅度差
2. 打包后杀毒软件频繁误报
3. 需要跨平台支持（Windows + Mac）

**迁移方案**：
- GUI 框架：customtkinter → PySide6/Qt
- 业务逻辑层：零改动，仅重写 GUI 层
- 架构：保留 multiprocessing.Process 多进程转写

## 八、技术栈

- **GUI 框架**：PySide6 (Qt)
- **转写引擎**：FunASR (SenseVoice + CAM++ + ct-punc)
- **录音模块**：PyAudioWPatch (WASAPI Loopback)
- **AI 服务**：MiMo API (云端)
- **配置管理**：JSON 文件
- **测试框架**：pytest
