## 一、全局样式 styles.py（P0 最高优先级）

### 1.1 【P0-BUG】全局 QFrame 嵌套边框

**问题**: `MAIN_STYLESHEET` 第 138-142 行的全局规则 `QFrame { border: 1px solid; border-radius: 8px }` 导致所有 QFrame（包括嵌套的）都出现边框，这是整个应用 UI 丑陋的根本原因。

**文件**: `src/gui/styles.py` 第 138-142 行

**修复**: 删除全局 QFrame 规则，改为 class 选择器：

```python
# 删除这段:
QFrame {{
    background-color: {C_CARD};
    border: 1px solid {C_BORDER};
    border-radius: 8px;
}}

# 改为（仅匹配带 cssClass="card" 的 QFrame）:
QFrame[cssClass="card"] {{
    background-color: {C_CARD};
    border: 1px solid {C_BORDER};
    border-radius: 8px;
}}
```

然后在代码中所有需要卡片效果的 QFrame 加上 `setProperty("cssClass", "card")`。例如:
- `home_page.py` 的 `rec_card`, `file_card`, `log_card`
- `settings_page.py` 的 `_create_group` 返回的 card
- `voiceprint_page.py` 的 `left_panel`, `right_panel`, `info_frame`
- `dialogs.py` 的各个弹窗中的 card

### 1.2 【P0-BUG】Primary 按钮颜色为红色（应为蓝色）

**问题**: `QPushButton[cssClass="primary"]` 使用 `C_ERROR`（红色 #C42B1C），而旧版主按钮是蓝色（C_ACCENT #0067C0）。

**文件**: `src/gui/styles.py` 第 163-173 行

**修复**:
```python
# 将 C_ERROR 改为 C_ACCENT，#A52318 改为 C_BTN_HOVER
QPushButton[cssClass="primary"] {{
    background-color: {C_ACCENT};
    color: white;
    border: none;
}}
QPushButton[cssClass="primary"]:hover {{
    background-color: {C_BTN_HOVER};
}}
```

### 1.3 【P1-BUG】ComboBox 下拉箭头消失

**问题**: `QComboBox::drop-down { border: none }` 移除了下拉箭头指示器，用户看不出这是可下拉的控件。

**文件**: `src/gui/styles.py` 第 207-209 行

**修复**:
```python
QComboBox::drop-down {{
    border: none;
    width: 20px;
}}
QComboBox::down-arrow {{
    image: none;
    border-left: 4px solid transparent;
    border-right: 4px solid transparent;
    border-top: 5px solid {C_TXT2};

```

### 1.4 【P1-缺陷】缺少 :pressed 伪状态

**问题**: 按钮没有 :pressed 状态，点击时无视觉反馈。

**修复**: 在 QPushButton 样式后追加:
```python
QPushButton:pressed {{
    background-color: #E0E0E0;
}}
QPushButton[cssClass="primary"]:pressed {{
    background-color: #004A8C;
}}
```

### 1.5 【P1-缺陷】常量定义了但未使用

**问题**: `FONT_SIZE_*`, `SPACING_*`, `BTN_SIZE_*` 等常量定义了，但各页面代码全部硬编码数值，没有引用这些常量。

**修复方向**: 后续统一替换各页面中的硬编码数值为常量引用（本次先解决功能和布局问题，此项可后续优化）。
